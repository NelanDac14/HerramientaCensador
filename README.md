# Herramienta Censador

## Descripción

Aplicación desarrollada para la gestión de clientes y seguimiento de visitas en campo, orientada a procesos de censado y distribución.

Permite registrar información de clientes, realizar seguimiento de visitas y optimizar rutas de trabajo.

---

## Objetivo

Facilitar el control y organización de clientes en campo, mejorando la eficiencia en procesos de venta, censado y logística.

---

## Tecnologías utilizadas

* Java (Android)
* SQLite
* XML
* Android Studio

---

## Funcionalidades

* Registro de clientes
* Gestión de visitas
* Seguimiento de rutas
* Almacenamiento local de datos (SQLite)

---

## Aprendizajes

* Desarrollo de aplicaciones móviles en Android
* Manejo de bases de datos locales
* Diseño de interfaces con XML
* Implementación de lógica de negocio real
